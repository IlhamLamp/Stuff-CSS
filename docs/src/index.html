Usage

Works without package manager or dependencies ðŸ™‚!

There are 4 ways to get started with stuff css:

Install manually

Download stuff and link /css/stuff.min.css in the <head> of your website. 

<link rel="stylesheet" href="css/stuff.min.css">

Install from CDN

Alternatively, you can use unpkg CDN to link stuff css 

<link rel="stylesheet" href="https://unpkg.com/stuffcss@latest/css/stuff.min.css">

Install with NPM

npm install stuffcss

Install with Composer

composer require stuffcss/stuff

Starter HTML template:

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <link rel="stylesheet" href="css/stuff.min.css">
    <title>Hello, world!</title>
  </head>
  <body>
    <main class="container">
      <h1>Hello, world!</h1>
    </main>
  </body>
</html>
