Breadcrumb

With aria-label="breadcrumb", you can turn a nav into a breadcrumb.

  • Home
  • Category
  • Page
<nav aria-label="breadcrumb">
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">Category</a></li>
    <li>Page</li>
  </ul>
</nav>
