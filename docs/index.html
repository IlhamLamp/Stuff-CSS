    Installation

    ✧*｡٩(ˊᗜˋ*)و✧*｡

    There are 3 ways to get started with Stuff:

    Install manually

    Download latest stuff and link /css/stuff.min.css

    <link rel="stylesheet" href="css/stuff.min.css">
    <!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <link rel="stylesheet" href="css/stuff.min.css">
    <title>Hello, world!</title>
  </head>
  <body>
    <main class="container">
      <h1>Hello, world!</h1>
    </main>
  </body>
</html>

    Install from CDN

    Alternatively, you can use unpkg CDN to link stuff css

    <link rel="stylesheet" href="https://unpkg.com/stuffcss@latest/css/stuff.min.css">

    Install with NPM

    npm install stuffcss
